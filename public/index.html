Slim Pickins is what I consider to be a minimal 'core' theme with everything I need to quickly develop new themes for Hugo. It is my first Hugo theme and is a port of my original Jekyll theme.

Basic features include:

  • Javascript free
  • Made for Hugo
  • Made with a minimal custom css grid + flexbox
    • 5 defined CSS grid areas: header, nav, main, aside, and footer.
    • Unique CSS grid templates for specific pages: homepage, about page, and singles.
  • Custom Sidebars
    • Recent Posts
    • Post Categories (with post count)
  • Responsive navigation menu
  • Optional full-width banner
    • Set a site-wide default
    • Optionally customize per page
  • Sticky footer
  • Minimal IcoMoon social fonticons included
  • Less than 600 lines of CSS
